# A Comprehensive Survey on Post-Quantum Cryptography Migration

**Venue:** IEEE Communications Surveys & Tutorials (target)  
**Status:** Manuscript under preparation (Phase B reconstruction)

## Overview

This repository contains the full reproducible research artifact for a structured narrative survey on post-quantum cryptography (PQC) migration across six critical infrastructure domains: TLS/Web PKI, IoT/Embedded, ICS/OT, Blockchain, and VPN/ZTNA.

The survey synthesizes 72 peer-reviewed publications and 39 normative/standards documents (111 sources total), proposing a Migration Readiness Classification (MRC) framework to prioritize PQC adoption.

## Reproducing the Figures

```bash
pip install -r requirements.txt
python figures/generate_all_figures.py
```

All figures read from `data/corpus_peer_reviewed.csv` and verified benchmark sources. No synthetic or hardcoded data.

## Running the MRC Scorer

```bash
# Score all six domains (reference year 2026)
python mrc_scorer.py --all --reference-year 2026

# JSON output for paper table
python mrc_scorer.py --all --reference-year 2026 --json
```

## Corpus Composition

- **72** peer-reviewed publications (screened and classified)
- **39** normative/standards documents (NIST FIPS, RFCs, government mandates)
- **9** duplicates removed from 120-record working list
- Corpus split generated by `data/split_corpus.py` — fully reproducible

This is a **structured narrative survey**, not a PRISMA systematic review.

## Key Standards Covered

- ML-KEM (FIPS 203), ML-DSA (FIPS 204), SLH-DSA (FIPS 205), FN-DSA (FIPS 206)
- HQC (selected March 2025 for standardization)
- EO 14412 (June 22, 2026) and OMB M-26-15 (June 24, 2026)
